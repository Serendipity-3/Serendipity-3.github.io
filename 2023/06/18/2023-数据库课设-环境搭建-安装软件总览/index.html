2023 数据库课设 环境搭建-安装软件总览

用到的工具

电脑两台

腾讯云 Ubuntu22.04 LTS（只要是云服务器就行） : 租的一台可以24小时运行的电脑，OS 为 Ubuntu 属于 linux

自己的电脑

编程语言软件

node.js: 运行 javascript 让你用 Vue3 的软件，Vue3 可以编译生成网页

jdk : 运行 java 程序让你用 SpringBoot 的软件

用到的工具软件

小工具

git：管理自己写的代码，可以管理代码的版本，万一写错可以回退，还用来上传代码到 gitee.com 上

maven：java 代码包的管理软件，方便用别人写好的代码包并管理代码包版本，基本在 Idea 中用到

**nvm ( Node Version Manager)**：巨方便，管理不同版本的 node 。有时候用别人的代码还要 node 低版本，有了它直接运行命令切换就行

大工具

nginx : 处理用户的浏览器请求，将网页( HTML CSS JavaScript) 发给用户的 web 服务软件
postgresql : 存储项目数据的数据库软件
redis : 缓存 postgresql 的数据，加速读写数据的数据库软件

idea：方便写 java 代码的软件
